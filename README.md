# NoKing — открытый протокол честного обмена

**No Kings. No Middlemen. No Fakes. Only Code.**

---

## Что это?

**NoKing** — это децентрализованный протокол для любых двусторонних соглашений об обмене ценностями.

Любая ситуация, где:

- **Сторона А** передаёт что-то (товар, услугу, время, информацию, право, доступ, ресурс).
- **Сторона Б** передаёт что-то взамен (деньги, другой товар, услугу, доступ, голос, долю).

**NoKing делает этот обмен честным, безопасным и прозрачным — без посредников.**

---

## Примеры использования

| Тип обмена | Сторона А отдаёт | Сторона Б даёт взамен |
|------------|------------------|-----------------------|
| Товары | Физический товар | Деньги (USDC / DAI) |
| Услуги | Часы работы, результат | Оплата за услугу |
| Фриланс | Выполненный проект | Оплата поэтапно |
| Аренда | Доступ к жилью/технике | Арендная плата |
| Подписка | Ежемесячный доступ | Регулярный платёж |
| Обучение | Знания, курс | Оплата за обучение |
| Интеллектуальная собственность | Права на использование | Роялти или паушал |
| Голоса | Голос в DAO | Вознаграждение за участие |
| Время | Консультация/менторство | Почасовая оплата |
| Доступ | К закрытому сообществу | Вступительный взнос |
| Энергия | Электричество / мощность | Оплата за ресурс |
| Данные | Анонимные данные для анализа | Вознаграждение за данные |
| Крипто-актив | Токен, NFT | Другой токен / стабильная монета |
| Бартер | Любая ценность | Любая другая ценность |

## Наши принципы

### 1. Клиентоориентированность
Удобно всем участникам. Интерфейс простой. Онбординг — быстрый.

### 2. Стабильность
Система не падает. Средства не теряются. Контракты зааудированы.

### 3. Кодекс чести
Правила игры прозрачны. Никаких скрытых алгоритмов. Честность — основа.

### 4. Народная демократия без фейков
Решения принимаются реальными участниками. Без ботов и олигархов.

## Что мы даём

| | Монополии (Озон, ВБ) / Посредники | NoKing |
|--|----------------------------------|--------|
| Комиссия | 15–30% | 0.5–1% |
| Рейтинг/репутация | Принадлежит платформе | Принадлежит участнику |
| Данные | Платформа продаёт | Принадлежат пользователю |
| Блокировка | Мгновенная, без объяснения | Только по решению сообщества |
| Арбитраж | Служба поддержки (на стороне платформы) | Сообщество (по правилам) |
| Алгоритмы | Непрозрачные | Открытые |
| Управление | Совет директоров | DAO (голосование участников) |
| Типы обмена | Только товары | Любые ценности (товары, услуги, время, информация, права, голоса) |
