# Глоссарий NoKing

| Термин | Определение |
|--------|-------------|
| **Сторона А** | Участник, который передаёт ценность (товар, услугу, время, информацию, право, доступ, ресурс) |
| **Сторона Б** | Участник, который получает ценность и передаёт что-то взамен |
| **Соглашение (Agreement)** | Смарт-контракт между Стороной А и Стороной Б, описывающий условия обмена |
| **Исполнение (Fulfillment)** | Факт выполнения обязательств одной из сторон |
| **Подтверждение (Confirmation)** | Факт подтверждения получения ценности второй стороной |
| **Эскроу (Escrow)** | Смарт-контракт, блокирующий средства до подтверждения исполнения |
| **Спор (Dispute)** | Ситуация, когда стороны не могут договориться об исполнении |
| **Арбитраж (Arbitration)** | Процедура разрешения спора сообществом |
| **DAO** | Децентрализованная автономная организация — коллективный орган управления |
| **Рейтинг (Rating)** | Числовой показатель репутации участника (0–5) |
| **Отзыв (Review)** | Текстовое мнение участника о другой стороне |
| **Кодекс чести (Code of Honor)** | Набор правил поведения для всех участников |
| **Голосование (Vote)** | Процедура принятия решений в DAO |
| **Форк (Fork)** | Разделение протокола на две независимые ветви |
